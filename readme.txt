-------------------------------------УСТАНОВКА----------------------------------------
1) Перейдите на страницу входа в аккаунт на сайте. Откройте панель разработчика в браузере chrome (f12), выберите вкладку network и фильтр XHR. После входа в аккаунт найдите XHR ответ от сервера (login). Скопируйте его (Copy as cURL bash). Перейдите на сайт curl.trillworks.com и вставьте данные. Заметим, что должен быть выбран язык python. После копирования вставляем эти данные в наш главный файл (main.py) вместо пустых header и data.
2) Запустите файл install.bat для настройки окружения.
3) Запустите файл launch.bat для запуска самой программы. В последующие разы открывать этот файл для запуска программы.
